Guia de Ícones para PWA - AZUOS TECH

Este documento contém uma lista de todos os ícones necessários para a implementação completa do Progressive Web App (PWA) para o site AZUOS TECH.

Nota: Todos os ícones devem ser criados a partir do logotipo da AZUOS TECH em formato PNG transparente ou com fundo sólido quando necessário.

Lista de Ícones Necessários

Nome do Arquivo Tamanho Propósito
favicon.ico 32x32 Favicon para navegadores
apple-touch-icon.png 180x180 Ícone para dispositivos iOS
icon-192x192.png 192x192 Ícone PWA para Android
icon-512x512.png 512x512 Ícone PWA grande para Android
ms-icon-70x70.png 70x70 Pequeno ícone para Windows
ms-icon-150x150.png 150x150 Ícone médio para Windows
ms-icon-310x310.png 310x310 Ícone grande para Windows
safari-pinned-tab.svg SVG Ícone para abas fixadas no Safari

Diretórios

Todos os ícones devem ser colocados no seguinte diretório:

c:\Users\azuos\SITE-AURORA\images\icons\

Próximos Passos

  1. Criar todos os ícones seguindo as dimensões especificadas
  2. Atualizar o manifest.json com os caminhos corretos para os ícones
  3. Adicionar as meta tags necessárias no head do HTML
